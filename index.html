Draw Figures

Привет! Здесь ты можешь нарисовать одну или сразу несколько фигур. Чтобы это сделать, нужно ввести строку, содержащую название фигуры, координаты, и параметры отрисовки: такие как цвет и заливка Линии и заливка могут быть как rgb, так и rgba. Если данны параметры не введены, то по умолчанию линии черные, а заливка прозрачная.

  • Названия фигур:

    line - линия
    rectangle - прямоугольник
    triangle - треугольник
    circle - круг
    ellipse - элипс

  • Ввод координат:

    -p [..., ...]

  • Цвет линии:

    -с rgb(..., ..., ...) или -с rgba(..., ..., ..., ...)

  • Цвет заливки:

    -b rgb(..., ..., ...) или -b rgba(..., ..., ..., ...)

Примеры:

line -p [50, 50] [100, 100] -c rgb(255, 0, 0)
rectangle -p [50, 50] [100, 100] -c rgb(0, 255, 0) -b rgba(255, 0, 0, 0.3)
triangle -p [50, 50] [50, 100] [100, 100] -c rgb(0, 0, 255) -b rgba(255, 0, 0, 0.3)
circle -p [75, 75] -r 25 -c rgba(255, 0, 0, 0.8) -b rgba(0, 255, 0, 0.3)
ellipse -p [75, 75] -r1 50 -r2 25 -c rgb(0, 255, 0) -b rgba(255, 0, 0, 0.3)
